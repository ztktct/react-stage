# React

## Info
`master` 分支提供基础脚手架，标配`react` + `webpack` + `babel`，其他分支请查看对应`README`文件

## Command
```bash
 $ npm run start # 启动项目
 $ npm run build # 编译打包
```

## Feature
- [x] 支持ES2015+
- [x] 支持SASS预处理器
- [x] 支出部分CSS3+新特性
- [x] 支持组件级热更新
- [x] 支持文件hash，有效解决缓存问题
- [x] ESlint代码规范检测
- [x] 分离公共依赖代码与业务代码
